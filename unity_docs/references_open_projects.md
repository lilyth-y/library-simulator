# Open Source Production Game References

아래는 코드가 공개된 프로덕션 게임(또는 장기 운영 오픈소스 게임) 중
경영/시뮬레이션/상호작용 시스템 참고에 도움이 되는 사례들이다.
장르가 동일하지 않더라도 시스템 설계 관점에서 참고할 가치가 있다.

## 1) OpenRCT2
- 테마파크 경영 시뮬레이션
- 경영 UI, 고객 흐름, 시설 운영 구조 참고

## 2) OpenTTD
- 교통/물류 경영 시뮬레이션
- 경제 루프, 시간 진행, 맵 관리 구조 참고

## 3) 0 A.D.
- RTS 게임
- AI, 유닛 상호작용, 빌드 파이프라인 참고

## 4) Minetest
- 샌드박스/크래프팅
- 인게임 상호작용, 모드 구조 참고

## 5) Battle for Wesnoth
- 턴제 전략
- UI/UX 구조, 데이터 주도 설계 참고

## 6) SuperTux
- 2D 플랫폼
- 아트 파이프라인 및 커뮤니티 운영 참고

## 참고 메모
- 위 프로젝트들은 공개 코드가 존재하며,
  시스템 설계와 구조를 참고하는 용도로 활용 가능하다.
- 도서관 경영과 직접적인 주제 일치는 낮으므로,
  UI/경제/상호작용 시스템 중심으로 참고하는 것을 권장한다.

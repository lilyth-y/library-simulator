# Content and Asset Specification

## 1) 레벨 구성
### 필수 구역
- 입구/로비
- 반납 구역
- 서가 구역(카테고리 3종)
- 수리대/정비 구역
- 직원 공간(확장 해금용)

### 동선 원칙
- 입구에서 모든 주요 구역이 한눈에 보이도록 배치
- 상호작용 빈도가 높은 오브젝트는 동선 중앙에 배치

---

## 2) 에셋 리스트(초기)

### 환경
- 모듈형 벽/바닥/기둥(로우폴리)
- 창문/문/조명

### 가구/소품
- 서가(작/중/대)
- 책상/의자
- 반납함
- 수리대
- 검색 PC/검색대
- 북트럭(책 운반 카트)
- 계단/난간(층 확장용)
- 안내판/사인
- 식물/장식품

### 책
- 기본 책 3~5종(색상 변형)
- 라벨/바코드 파츠(수리 연출용)

---

## 3) 프리팹 규칙
- 모든 프리팹은 Pivot이 바닥 중앙
- 스케일 기준: 1유닛 = 1m
- 책 기본 크기 예시: 0.2 x 0.03 x 0.14
- 콜라이더는 Box Collider 기본

---

## 4) LOD/퍼포먼스
- 서가/가구: LOD 2~3단계
- 책: 단일 LOD(로우폴리 유지)
- 텍스처 크기 권장: 512 또는 1024

---

## 5) 머티리얼 규칙
- URP Lit
- 베이스 컬러 + 간단한 AO만 사용
- 금속/거칠기 최소화(카툰 감성 유지)

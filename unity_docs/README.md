# Unity Docs - Library Town (Working Title)

이 디렉토리는 Unity 기반 개발을 위한 문서 모음이다.
목표는 두근두근타운 모티브의 아기자기한 도서관 경영 게임을
PC/모바일에서 동일한 플레이 감성으로 구현하는 것이다.

## 문서 구성
1) game_design.md
   - 하이 컨셉, 코어 루프, 플레이어 경험, MVP 범위
2) system_specs.md
   - 상호작용, 물리, AI, 업무/퀘스트, 데이터 구조 상세 명세
3) ui_ux.md
   - 화면 구성, 입력 흐름, UI 동선, 피드백 규칙
4) content_assets.md
   - 레벨 구조, 에셋 목록, 프리팹/머티리얼 규칙
5) technical_plan.md
   - Unity 프로젝트 구조, 패키지, 성능/빌드 가이드
6) references_open_projects.md
   - 참고 가능한 오픈 소스 프로덕션 게임 목록

## 범위 가정
- 플랫폼: PC + 모바일
- 아트: 스타일라이즈, 로우폴리곤
- 핵심 재미: 물리 상호작용 + UI 기반 경영 + 캐릭터 상호작용
- 엔진 경험 없음 기준으로 개발 난이도를 낮춘 설계

## 문서 업데이트 원칙
- 기능 확정 시 해당 문서를 먼저 갱신한다.
- 실제 구현과 문서의 차이가 발생하면 문서를 우선 수정한다.
